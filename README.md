# 케이뱅크 포인트허브 개편안 (UI 목업)

PM 제안 · 데이터 기반 매출 증대 개편안을 위한 UI 목업.

## 배포 페이지

- [index.html](./index.html) — 진단 요약 + change list + 목업 링크
- [as-is.html](./as-is.html) — 현재 라이브 지면 재현
- [to-be.html](./to-be.html) — 개편안 반영 화면

## 배경

케이뱅크 매체의 매출 87%가 display 광고 2개 unit에 집중된 구조를 진단하고, 유저 참여 깊이와 매체 특성에 맞는 IA로 재구성.

- 분석 기간: 2026-03-25 ~ 2026-04-24 (30일)
- 기준 매출: ₩661,662,000
- 일 평균 ImpDAU: 303,929명
- 업계 내 eCPM 순위: 1위 (2,837원)

## 6개 Change 요약

| # | 개편 | 데이터 근거 | 예상 임팩트 |
|---|------|------------|------------|
| 1 | 매일버튼 → "이어받기" 오버레이 | eCPM 10,836원 unit, CTR 99.8%, 후속 매출 없음 | +8~12% |
| 2 | 오늘의 용돈 1개 → 3개 캐러셀 | Feed+오늘의용돈 매출 62.6% | +15~25% |
| 3 | AI 맞춤 미션 상단 고정 슬롯 | action_heavy CVR 58.9%, 매출 비중 10.7% | action_heavy 18~22% |
| 4 | PLAYHUB + Post-game 광고 | 노출 2,191만, 매출 0원 | +3,500만원 |
| 5 | 콘텐츠 → CPA 연계 | 콘텐츠 unit eCPM 2.82원 | eCPM 70배 개선 |
| 6 | 보험 → "금융 꿀팁" 리네이밍 | 보험 CVR 6.6% (최저) | CVR 2배 |
